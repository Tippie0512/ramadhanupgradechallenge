# Ramadhan Upgrade Mobile App

Aplikasi mobile sederhana untuk kompetisi Ramadhan Upgrade 2026, dibuat menggunakan Kotlin untuk platform Android.

## Fitur

- **Salam Ramadhan**: Tampilan salam saat aplikasi dibuka
- **Progress Ramadhan**: Tombol untuk melihat hari keberapa Ramadhan (contoh: hari ke-15)
- **Doa Buka Puasa**: Tombol untuk menampilkan doa buka puasa

## Tema Ramadhan

Aplikasi ini mengangkat tema Ramadhan dengan:
- Salam Assalamu'alaikum
- Informasi progress bulan Ramadhan
- Doa untuk buka puasa

## Cara Menjalankan

1. Buka project ini di Android Studio
2. Build dan jalankan di emulator atau device Android
3. Aplikasi akan terbuka dengan tampilan utama

## Teknologi

- Bahasa: Kotlin
- Framework: Android SDK
- Minimum SDK: API 21 (Android 5.0)
